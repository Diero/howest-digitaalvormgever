## [1.8.3]
- Added a small maintenance release containing documentation and metadata updates and minor snippet fixes.
- Translated and cleaned up `README.md` (English).
- Fixed broken links and typos in the README.
- Corrected snippet prefix bug (separated `mct-random` and `wdd-random` into distinct prefixes instead of a single comma-separated string).
- Deprecated or moved `OLD-` snippets to reduce IntelliSense noise.
- Improved `package.json` metadata: added `keywords`, `bugs` URL, `homepage`, repository object and `license` field; added packaging/validation scripts.
- Updated `engines.vscode` minimum to a newer baseline.
